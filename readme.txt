== Description ==

This plugins user for change comment avatar. First it author avatar removed and after the new look of avatar like skype avatar with short name.This plugins allows only comment author avatar. 

How to use this plugin?

1. Download this Free Comment Name Avatar WordPress Plugin from above link.
2. Connect to your WordPress dashboard (wp-admin) and navigate to Plugins >> Add New Plugin >> Upload Plugin >> Now upload the downloaded (Comment-Name-Avatar.zip) file >> Click on  Install Now.
3. Once you install this plugin successfully, click on  Activate Plugin .


Now you visit comment area in your theme and look comment author avatar changed.


== Installation ==

**Installation**

1. Upload the entire 'Comment-Name-Avatar' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
